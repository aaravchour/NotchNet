The heavy core is a rare block that can be used to craft a mace.


== Obtaining ==


=== Breaking ===
Heavy core can be mined with any tools or an empty hand, but a pickaxe is the quickest way to mine it.


=== Loot ===
The only way to obtain a heavy core is to unlock an ominous vault using an ominous trial key. Each vault has a 7.5% chance of dropping a heavy core.


== Usage ==
The only current known use for the heavy core is to craft a mace.
