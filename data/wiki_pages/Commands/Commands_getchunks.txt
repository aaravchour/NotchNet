Syntax
getchunks <dimension: string>
Example usage:
getchunks overworld
Returns an array of arrays. Each of the arrays contains an x and z coordinate.
e.g.
[\n' +
      '   [ 80, -3 ],\n' + 
      '   [ 3, -4 ],\n' +  
      '   [ 78, -3 ],\n' + 
      '   [ 77, 6 ],\n' +  
      '   [ 76, -4 ],\n' +
