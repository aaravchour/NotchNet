Toggles the world to be able altered or cannot be altered.


== Syntax ==
immutableworld [value: Boolean]


== Arguments ==
value: Boolean: enum

Specifies the value to set the world to be able altered.
Must in between true or false.


== Result ==
Fails if the argument is not specified correctly.
On success, if true the world cannot be altered, but doors, levers, etc. can still be used, and vice versa if set to false. (This only applies to non-operator player). If value: Boolean is not specified, displays current value.
