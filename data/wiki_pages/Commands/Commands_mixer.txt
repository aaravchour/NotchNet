Controls Mixer interactivity.


== Syntax ==
mixer <stop: string>
mixer <scene: string> <sceneName: string>
mixer <start: string> <version: int> [shareCode: string]


== Arguments ==
start

Starts the interactive board.
version

Specifies the version ID of the player's board. (for example, 000001)
shareCode

Specifies the share code of the player's board. (for example, 0000aaaa)
scene

Changes scene.
sceneName

Specifies the name of the scene the player wants to change to. (for example, default)
stop

Specifies that the player wants to stop the interactive board.


== Result ==
start mode:

The MixPlay board begins.
scene mode:

The scene changes.
stop mode:

The MixPlay board shuts down.
Fails if the arguments are not specified correctly.


== Interactive Board Setup ==
Login to your account at the mixer interactive lab.
Click on the MixPlay Projects button.
If you haven't made a board before, click Let's Create One. If you have already made a board, click the + button in the top right.
Name the board whatever you would like and set the game to Other.
Click on Build.
The default scene is the main one that is loaded upon starting with the /mixer command. This cannot be renamed.
In Controls, click the + to add a button.
Name it, click Button, then click Add.
Click on the > and put some text in. This is shown on the button. If you want to use an image, put a link of the image in Background Image.
If wanted, fill in the other boxes. Note that only the text is required.
Go to the meta properties and click + Add Property.
Name it Command (it must be Command or it won't work) and leave the type as Text.
In the Value area, type in your command. (for example, /summon zombie)
Click the X and hold the =. While holding it, drag it where you would like it to be on the board.
If wanted, resize it using the dots on the side.
Now to use it. Click the share icon in the top right corner and click Anyone with the versionId and code can play.
In Minecraft Bedrock, type /mixer start VERSIONID SHARECODE.
If done successfully, you can click the buttons on your stream. When the button is pressed, the command is executed if set up correctly.
