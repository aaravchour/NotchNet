The void quiver is an artifact that can be found within the echoing void DLC of Minecraft Dungeons.


== Obtaining ==
Missions
Broken Citadel
Merchants
Village Merchant
Gift Wrapper
Mystery Merchant
Luxury Merchant


== Usage ==
The void quiver grants the hero arrows tipped with the voided effect, which allows multiplying damage once inflicted onto a target mob.


=== Properties ===
X ranged damage
25 second cooldown


== Void arrow ==

A void arrow is an arrow tipped with the voided effect that can be found within the echoing void DLC of Minecraft Dungeons.

== Data values ==
