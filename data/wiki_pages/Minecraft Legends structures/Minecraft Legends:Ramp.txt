A ramp is a structure that creates stairs or bridges within Minecraft Legends.


== Obtaining ==
Ramps can be placed by picking a starting point and an end point in placement preview mode. One build allay will be used to create the entire ramp, but each column in the ramp will cost three wood. It can also be built on netherrack.


== Usage ==
The ramp can create stairs and bridges, which can be used to overcome obstacles.
